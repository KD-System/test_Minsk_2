# Погода в Минске

Простой сайт с текущей погодой и прогнозом для Минска. Данные — [Open-Meteo](https://open-meteo.com/) (без API-ключа).

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Открыть: http://localhost:5000

## Возможности

- Текущая температура, ощущается как, описание
- Ветер, влажность, давление, осадки
- Восход и закат
- Почасовой прогноз на 24 часа
- Прогноз на 7 дней
- Адаптивный дизайн, дневная/ночная тема
